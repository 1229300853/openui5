Add a new project

Follow these steps to add a new project to the list above:

  1. Start by going to our Git repository SAP OpenUI5.
  2. Login or register on GitHub.
  3. Fork the OpenUI5 project to your repository.
  4. Switch to the gh-pages branch.
  5. Select the javascripts folder and open the relatedProjects.js.
  6. At the beginning of the file you will find the oFeaturedProjects object. Here you have to add a new entry. It should look like this example: 
    this.oFeaturedProjects = {
  "UI5CustomControl": {
	...
  },  "<keyWord>": {
    "name": "<official name of your project>",
    "description": "<short description of your project>",
    "documentaionLink": "<link to the documentation>",
    "owner": "<the owner of the project>",
    "license": "<license of your project>"
  }
};
  7. Commit your changes.
  8. Make a new pull request.
  9. Your change will be reviewed and the project is listed.
