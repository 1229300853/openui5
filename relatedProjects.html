Want to upload your own project? Click on and follow the instructions

  • Start by going to our Git repository SAP OpenUI5.
  • Login or register on GitHub.
  • Click on the fork button in the top right corner of the screen.
  • Click on the branch button on the left and then select the gh-pages branch.
  • Select the javascripts folder and open the relatedProjects.js.
  • Enable the edit button.
  • At the beginning of the file you will find the oFeaturedProjects object. Here you have to add a new entry. It should look like this example: 
    								this.oFeaturedProjects = {
									"Ui5Strap": {
										"name": "Ui5Strap",
										"decription": "-",
										"githubLink": "https://github.com/pks5/ui5strap",
										"documentaionLink": "http://www.ui5strap.com/manual/",
										"owner": "-",
										"license": "Apache 2"
									},
									"UI5CustomControl": {
										"name": "OpenUI5 Custom Controls for charting",
